This website (the "Site") is designed to respect the visitors' privacy, while still providing a secure and functional experience.

## Data Processed

When people visit the Site, the Site's infrastructure provider <NewWindow href="https://www.cloudflare.com/">Cloudflare</NewWindow> processes basic request data to deliver and secure the Site (such as IP address, request headers, requested URL, and security-related metadata). Cloudflare notes this metadata often includes IP addresses and is processed for a limited period (see Cloudflare's <NewWindow href="https://www.cloudflare.com/trust-hub/gdpr/">GDPR / data processing overview</NewWindow>). The Site uses <NewWindow href="https://www.cloudflare.com/web-analytics/">Cloudflare Web Analytics</NewWindow> to understand overall traffic and performance of the Site. Cloudflare states its Web Analytics does not use cookies or localStorage and does not fingerprint individuals for the purpose of displaying analytics. Web Analytics is designed to collect minimal performance/timing metrics via a beacon. The Site only uses this information to:

1. Serve the Site
2. Maintain security and prevent abuse
3. Understand aggregate traffic and performance

## Cookies and Tracking

This Site does not use marketing or advertising cookies. The Site uses <NewWindow href="https://www.cloudflare.com/web-analytics/">Cloudflare Web Analytics</NewWindow>, which Cloudflare states does not use cookies or localStorage and does not fingerprint individuals for the purpose of displaying analytics. If Cloudflare security features are triggered (for example, bot protection or challenge pages), Cloudflare may set strictly necessary cookies to help protect the Site. See Cloudflare's documentation on <NewWindow href="https://developers.cloudflare.com/fundamentals/reference/policies-compliances/cloudflare-cookies/">Cloudflare cookies</NewWindow> for details.

The Site shares data with service providers, Cloudflare, only as needed for content delivery, security, and analytics. The Site does not run behavioral advertising and does not track visitors across third-party websites over time. Browser "Do Not Track" signals are not used to change behavior on this Site (it still processes basic request/security data required to operate the Site).
